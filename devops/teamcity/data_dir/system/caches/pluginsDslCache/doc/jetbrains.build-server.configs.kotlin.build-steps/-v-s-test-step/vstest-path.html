vstestPath

var vstestPath: String?

A path to test engine. TeamCity detects test engine installation on the agent, to run the detected engine use the following paths:

  • VSTest 2019 - "%teamcity.dotnet.vstest.16.0%"
  • VSTest 2017 - "%teamcity.dotnet.vstest.15.0%"
  • VSTest 2015 - "%teamcity.dotnet.vstest.14.0%"
  • VSTest 2013 - "%teamcity.dotnet.vstest.12.0%"
  • VSTest 2012 - "%teamcity.dotnet.vstest.11.0%" Or specify a custom path to the test engine on a build agent machine.
