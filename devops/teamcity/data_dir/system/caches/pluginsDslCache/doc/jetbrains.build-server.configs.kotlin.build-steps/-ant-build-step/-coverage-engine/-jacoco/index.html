classLocations

 var classLocations: String?

Newline-delimited set of path patterns in the form of +|-:path to scan for classfiles to be analyzed. Excluding libraries and test classes from analysis is recommended. Ant like patterns are supported.

excludeClasses

 var excludeClasses: String?

Newline-separated patterns for fully qualified class names to be excluded from the coverage. Exclude patterns have priority over include patterns.
